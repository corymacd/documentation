The Amazon S3 Sink connector exports data from Kafka topics to S3 objects in JSON format.

## Prerequisites

Before you create an S3 sink connector in the Redpanda Console, complete the following tasks:

* In the AWS management console, create an S3 bucket associated with your AWS account ID, and an IAM policy to govern access to the bucket.
* In the Redpanda Console, create one or more topics.
